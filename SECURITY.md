# Security Policy

## Reporting a Vulnerability

We take the security and privacy of user data very seriously. Please report security vulnerabilities by opening a [Private Vulnerability Report](https://github.com/reponomics/PolicyChecks/security/advisories/new). If, for some reason, you are unable to do so, you may also contact `security@reponomics.org`.

Your report will receive immediate attention, and follow-up steps will be discussed within 14 days at most.
