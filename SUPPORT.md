# Support

PolicyChecks is a free badge service for transparently reporting GitHub repository administration settings.

## Setting Up Badges

For badge setup instructions, supported badge names, endpoint formats, and Markdown examples, see the [README](README.md).

## Service Issues

Please open a GitHub issue for problems that may affect the service generally, including:

- Badge endpoint errors or unexpected response formats.
- Suspected service outages or degraded availability.
- Inaccuracies in how PolicyChecks reports a supported repository setting.
- Documentation problems that make setup or interpretation unclear.

Public issues are preferred for general service defects because they make the problem, discussion, and resolution visible to other users.

## Repository-Specific Questions

For questions, concerns, or complaints about the status of a specific repository badge, contact:

```text
support@reponomics.org
```

Email is preferred for repository-specific questions because the details may depend on the repository, installation state, or administrative settings involved.

## Security Issues

Please do not report security vulnerabilities through public issues or the general support address. See [SECURITY.md](SECURITY.md) for vulnerability reporting instructions.
